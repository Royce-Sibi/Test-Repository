Acheive Console Working(Project)

Visual Code Studio Is A Coding Platform With A Variety Of Extenstions. There's Not Really Anything To Set It Up. You'll Figure IT Out Yourself.

To Enter The Console Screen, Right Click (RMB) And Press Inspect. You'll Start Off In The Elements Tab. So Click The Console Tab At The Top. To Print Something Into The Console, Use The "Console.log" Cmd. The Use Of The Console Is To See What Is Going On.
